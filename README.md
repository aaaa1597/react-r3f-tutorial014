# ポイント
## 光源にcastShadowの属性をつける.
directionalLight(指向性光源(太陽光))、pointLight(点光源)、spotLight(線光源)にShadow属性をつける。

## 物体の方にreceiveShadowとcastShadowをつける。
- 影を受け付ける設定 ... receiveShadow
- 影を出す設定 ... castShadow
